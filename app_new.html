Basic HTML Page

What's in a basic HTML5 page?

The building block of all webpages made with HTML are tags.

Tags serve different purposes - they can be used to add text to a page, such as the text you're reading now.

You'll have the chance to learn about many different tags when taking the Tag Quiz at the main menu.

For now, though, the focus will be on some basic tags that allow you to specify your page's text, as well as the page title, and the general structure of the HTML5 page.

How Tags Work

Most tags you will use or encounter follow these same rules: they will come in a pair of two, the opening tag, and the closing tag.

All tags, though, will consist of the name of that tag, and two angle brackets surrounding that name. Here's an example:

<title> My Page Title </title>

The name of the tag, which is 'title', indicates the Title tag, used to give a title to your page which will appear on your browser's tab and elsewhere. Some tags make use of text; in this case, 'My Page Title' would become the title of the page when viewed.

You can tell which of the two tags is the closing tag, as the closing tag in a pair will always start with a forward slash symbol ( / ).

HTML5 Page Structure

Now that you know how tags work, click 'Begin' on the bottom right of this screen to observe the general structure of a basic HTML5 page. You'll be able to drag and drop the tag blocks into their respective places to see how certain tags fit together.
