A beginner's resource in understanding web design

By combining a variety of resources into a mobile-geared app, HTML Primer allows users to gain a fast, working knowledge of how to write web pages - learning along the way on either their home computer, or a smartphone


Dynamic Quiz: Randomized questions with embedded links

Learn from right and wrong answers when you take the HTML Element quiz. This dynamic quiz randomly generates questions using Javascript and jQuery from a collection of JSON objects, with possible answers in different orders; wrong answers indicate what that element choice does, and even links to a page where the user can learn more about that element.
