Responsive Video Portal

front-end frameworks

back-end technologies

I used Bootstrap and Susy grids to implement a redesign of a video portal page for Louise Blouin Media. Using Bootstrap was a great way to get useful features and components out of the box, such as the navigation bar and grid system. Susy grids were also very useful, as the requirements for this responsive design called for a grid system that could display content on anywhere from three to two columns, or a single column on small devices. Aside from the advantages of Susy, using Sass helped organize the project by allowing me to separate the front-end styles into smaller stylesheets used to over-ride default formatting of Bootstrap components.

For the backend part of this portal, the PHP framework Symfony was used to write logic that would make requests to Brightcove’s Media API using a PHP wrapper. Registering the Media API service for use was easy, as Symfony provides a well-documented and predictable means of registering such services. A utility class containing categories and filtering functions was used to issue requests to Brightcove. Symfony’s default templating system, Twig, was used to create the templates used for rendering a single video, a grid of videos to watch, and small components like ‘Share’ buttons and links to related articles.
